# Resource Booking System

Рабочий прототип системы бронирования корпоративных ресурсов для курсового проекта.

## Описание
Приложение предназначено для:
- просмотра корпоративных ресурсов;
- добавления новых ресурсов;
- создания бронирований;
- просмотра журнала бронирований;
- отмены бронирований;
- проверки конфликтов по времени.

## Технологии
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- HTML / CSS / JavaScript

## Структура проекта
- `controller` — REST-контроллеры
- `service` — бизнес-логика бронирования
- `repository` — доступ к базе данных
- `entity` — сущности предметной области
- `dto` — объекты передачи данных
- `static` — веб-интерфейс

## Основной функционал
- каталог корпоративных ресурсов;
- создание бронирования;
- проверка пересечения временных интервалов;
- отмена бронирования;
- журнал бронирований.
